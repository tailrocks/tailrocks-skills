# Worked Plan-package Example

> Format anchor only. Do not execute this package: its repository and
> `example://` evidence are invented. Run `tailrocks-plan` against the live
> item instead.

This example traces one READY item through the planning artifacts:

1. [`roadmap/goal-live-status/README.md`](roadmap/goal-live-status/README.md)
   supplies settled intent.
2. [`research/goal-status-ipc/`](research/goal-status-ipc/) supplies vetted
   example evidence.
3. [`coverage.md`](plans/goal-live-status/coverage.md) inventories statements.
4. [`spec/`](plans/goal-live-status/spec/) turns them into contracts.
5. [`README.md`](plans/goal-live-status/README.md) orders three executable
   plans.
6. [`GOAL.md`](plans/goal-live-status/GOAL.md) hands the package to an
   autonomous loop.
